# 完整输出示例说明

## 📊 概述

本项目包含一个完整的输出示例文件，展示五兆占卜系统的所有功能和详细输出。

## 🚀 运行方式

```bash
cd kinwuzhao-ts
node examples/complete-output.mjs
```

  - 特殊标记（孤、虚、关、籥、将军）

## 💡 技术要点

### 1. 干支计算

使用 `lunar-typescript` 库计算精确的干支：

```javascript
const gz = ganZhi(2025, 6, 27, 11, 24);
// { year: '乙巳', month: '壬午', day: '丁卯', ... }
```

### 2. 节气计算

使用 `astronomy-engine` 库计算太阳黄经：

```javascript
const jq = findCurrentJieQi(2025, 6, 27, 11, 24);
// '夏至'
```

### 3. 五兆排盘

两种模式：
- **干支起盘**: 确定性计算
- **随机排盘**: 模拟实际占卜

```javascript
// 干支起盘（确定性）
const result1 = ganzhiPaiPan(gz, 0, jq, '五');

// 随机排盘
const result2 = fiveZhaoPaiPan(0, '夏至', '五', '甲子', '庚午');
```
